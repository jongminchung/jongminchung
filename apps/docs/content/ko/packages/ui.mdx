## 상태

`@jongminchung/ui` 1.0.0은 기존 소비자를 위해 API와 배포 계약을 유지하지만 신규 기능을 받지 않습니다. 새 UI는 Astryx를 사용하세요. 패키지를 제거하거나 1.0.0 공개 export를 바꾸는 변경은 이 마이그레이션 범위에 포함되지 않습니다.

## Astryx 마이그레이션

1. 앱 루트를 Astryx `Theme`과 `AppShell`로 감쌉니다.
2. reset, base, Neutral theme CSS 레이어를 명시합니다.
3. `Button`, `Badge`, `Card`를 화면 단위로 교체합니다.
4. Tailwind 토큰 의존을 제거한 뒤 키보드·다크 모드·반응형 동작을 확인합니다.

```tsx
import { Card } from "@astryxdesign/core/Card";
import { Theme } from "@astryxdesign/core/theme";
import { neutralTheme } from "@astryxdesign/theme-neutral";

export function App() {
  return (
    <Theme theme={neutralTheme} mode="system">
      <Card padding={4}>Migrated content</Card>
    </Theme>
  );
}
```

## 1.0.0 API

### 컴포넌트

- `Button`, `buttonVariants`, `ButtonProps`
- `Badge`, `badgeVariants`, `BadgeProps`
- `Card`, `CardHeader`, `CardTitle`, `CardDescription`, `CardContent`, `CardFooter`와 각 props 타입

### 디자인 토큰과 유틸리티

- `designSystem`, `DesignSystem`: Vercel 분석 기반 색상·타입·간격 객체
- `cn`: `clsx`와 `tailwind-merge`를 합친 클래스 병합 함수

### CSS subpath

`styles.css`, `tokens.css`, `baseline.css`는 1.0.0 소비자를 위해 유지됩니다. Astryx 앱에서는 가져오지 마세요.
